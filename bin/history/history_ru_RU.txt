История версий XFLTool
========================================================
0.0.4 alpha [15.01.12]
- проблемы и подсказки:
  - модуль полностью переписан
  - введены новые типы проблем - производительность и опрятность
  - улучшен вывод непрерывно повторяющихся из кадра в кадр проблем
  - добавлен ряд новых правил поиска проблем:
	- поиск временных шкал со слоями разной длины
	- поиск Shape'ов с внешней обводкой
	- поиск классических прямоугольных масок
	- поиск пустых слоёв
	- поиск неиспользованных маск
	- поиск неиспользуемого кода
	- поиск неиспользуемых ключевых кадров
	- поиск неиспользованных tween'ов
	- определение использования зарезервированных слов в качестве Instance Name'ов
	- определение использования нескольких монтажных каров (сцен)
	- поиск кода и меток не на пустых слоях
	- поиск неиспользуемых текстовых полей
	- поиск неверно выбранного типа текстовых полей
	- поиск в библиотеке элементов Graphic
	- поиск применённых фильтров с blurX или blurY не являющимися степенью 2
  
- анализ сжатия изображений:
  - вырезано всё, что могло вносить изменения в проект - теперь этот модуль служит только для анализа
  
- исправлены различные проблемы в UI и логические ошибки
- журналы событий теперь полностью текстовые - в угоду производительности и удобству
- добавлена страница Опции
- добвлена возможность выбора пути до Flash Pro
- изменён механизм обновлений
